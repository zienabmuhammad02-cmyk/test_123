Auto Text-to-Speech from Selection (English + Arabic)

Select text anywhere on the page and it will speak automatically. If your browser blocks audio, click once on the page to enable speech.

English text

Select any part of this paragraph. The page will automatically read the selected text out loud. Selecting new text will stop the previous speech and start the new one.

This uses the browser Web Speech API, so it is free and needs no backend.
